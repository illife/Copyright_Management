# 联盟链版权管理系统 - 前端项目说明

## 项目概述

基于 Vue 3 + Vite + Element Plus 构建的现代化版权管理系统前端，采用深色科技风格主题，提供完整的版权、授权、版税管理功能和区块链可视化。

## 技术栈

- **框架**: Vue 3 (Composition API)
- **构建工具**: Vite 5.0
- **UI组件**: Element Plus (深色主题定制)
- **图表**: ECharts 5 + vue-echarts
- **状态管理**: Pinia
- **路由**: Vue Router 4
- **HTTP客户端**: Axios
- **样式**: SCSS
- **日期处理**: Day.js

## 设计规范

### 颜色主题（深色科技风）

```scss
// 主色调 - 深蓝色系
--bg-primary: #0a0e27;        // 主背景
--bg-secondary: #131829;      // 次级背景
--bg-tertiary: #1a1f38;       // 三级背景
--bg-card: #1e2442;           // 卡片背景

// 强调色 - 科技蓝
--accent-primary: #00d4ff;    // 主强调色
--accent-secondary: #4facfe;  // 次强调色
--accent-success: #00ff9d;    // 成功色
--accent-warning: #ffa940;    // 警告色
--accent-danger: #ff5b5b;     // 危险色

// 文字颜色
--text-primary: #ffffff;      // 主文字
--text-secondary: #b8c7e0;    // 次级文字
--text-tertiary: #7d8ca8;     // 三级文字
```

### 设计特点

1. **深色主题**: 深蓝/深灰背景搭配亮色元素
2. **科技感**: 渐变色彩、发光效果、动画过渡
3. **严肃安全**: 政府网站风格，稳重大气
4. **响应式**: 支持桌面端和移动端
5. **可视化**: 丰富的图表和动画效果

## 注意事项

1. 确保后端服务运行在 `http://localhost:8081`
2. 首次使用需要注册区块链地址
3. Token默认存储在localStorage中
4. 所有请求都需要JWT认证
5. 生产环境请修改 `.env` 配置

## 开发建议

1. 使用Vue DevTools进行调试
2. 遵循Vue 3 Composition API规范
3. 组件命名使用PascalCase
4. 文件命名使用PascalCase（组件）或kebab-case（工具）
5. 使用SCSS编写样式，利用全局变量
6. 图表使用ECharts，参考官方文档
7. 响应式设计使用Element Plus的栅格系统

## 10. 响应式设计考虑
- **桌面端**：充分利用宽屏空间，显示更多信息
- **平板端**：侧边栏可折叠，内容区域自适应
- **手机端**：垂直布局，简化操作，重点功能突出

## 11. 特色功能页面
- **区块链浏览器**：可视化展示版权在链上的流转
- **版权追溯**：时间轴展示版权历史
- **智能合约管理**：可视化合约状态和执行情况
- **社区功能**：版权交易市场、创作者交流

这个设计既保证了功能完整性，又注重用户体验，特别是登录注册流程的友好性，让不同技术水平的用户都能轻松使用基于区块链的版权管理平台。
